# React

# 浅谈React数据流管理

  • 推荐理由: 为什么数据流管理如此重要？react的核心思想就是：UI=render(data)，data就是我们说的数据，render是react提供的纯函数，所以用户界面的展示完全取决于数据层。这篇文章希望能用最浅显易懂的话，将react中的数据流管理，从自身到借助第三方库，将这些概念理清楚。
