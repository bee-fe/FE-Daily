# SWR

SWR 是一个用来获取远程数据的 React Hooks，它的名字来自stale-while-revalidate，而stale-while-revalidate是RFC 5861提出的一种缓存失效策略。SWR 可以使得在页面页面渲染时先使用旧的数据渲染页面，同时发送请求获取新数据，然后再使用新的数据渲染页面。非常适用于一些数据很少修改，性能要求较高，数据准确性不敏感的情况。
