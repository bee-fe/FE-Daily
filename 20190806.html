# Qcharts 介绍与推荐

  • 投稿理由：QCharts 是一个拥有强大配置功能的图表库，在处理复杂图表时，它强大的配置项能让我们对绘制的图表进行全方位的掌控，并且它有着官方提供的React和Vue组件库，同时支持在微信小程序中使用，各个版本 api 基本一致，熟悉一个便可以贯通所有。使用它可以更好的完成数据展示任务，可以更好的提高我们的工作效率
